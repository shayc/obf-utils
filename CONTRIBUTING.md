# Contributing to OBF Utils

Thank you for your interest in contributing to OBF Utils! Your help is welcome and appreciated.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Clone your fork**:
   ```bash
   git clone https://github.com/shayc/obf-utils.git
   cd obf-utils
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Create your feature branch**:
   ```bash
   git checkout -b my-feature
   ```
5. **Make your changes** and add tests as needed.
6. **Run tests and lint** before submitting:
   ```bash
   npm test
   npm run lint
   ```
7. **Commit your changes**:
   ```bash
   git commit -am "Add feature/fix bug"
   ```
8. **Push to your fork** and submit a Pull Request (PR) to the main repository.

## Coding Standards

- Use TypeScript and follow the existing code style.
- Write clear, descriptive commit messages.
- Add or update tests for any code changes.
- Run `npm run lint` and `npm run format` to ensure code quality.

## Reporting Issues

- Use [GitHub Issues](https://github.com/shayc/obf-utils/issues) to report bugs or request features.
- Please provide as much detail as possible, including steps to reproduce, expected behavior, and screenshots if applicable.

## Code of Conduct

By participating, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions?

Open an issue or start a discussion on GitHub.

Thank you for helping make OBF Utils better!
